# Reza Kurniadi — Portfolio (Single-Page Scroll)

Pure HTML5 + CSS3 + Vanilla JS. No frameworks. Zero dependencies.

## Cara Pakai di VS Code
1. Unzip folder ini ke mana saja
2. Buka folder di VS Code
3. Install extension: Live Server (Ritwick Dey)
4. Klik kanan index.html → Open with Live Server
5. Browser buka otomatis ✅

## Deploy ke GitHub Pages (Gratis)
1. Buat repo baru di GitHub
2. Upload semua file ke root repo
3. Settings → Pages → Source: main branch → Save
4. Live di: https://namakamu.github.io/nama-repo/

## Tambahkan CV PDF
1. Buat CV di Canva / Word / Google Docs → export ke PDF
2. Rename file jadi: Reza-CV.pdf
3. Taruh di: assets/pdf/Reza-CV.pdf
4. Selesai — semua tombol download sudah mengarah ke sana!

│   └── sections.css    ← Style tiap section

│   └── app.js          ← Semua JS dalam 1 file bersih
├── assets/
│   ├── pdf/            ← Taruh Reza-CV.pdf di sini
│   ├── img/            ← Foto profil (opsional)
│   └── icons/
└── README.md
